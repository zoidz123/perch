Verdict

Charts must deliver the answer in 15 seconds

Recommended One screen

Problem / Findings

  • Long charts hide the decision.
  • Background repeats what the boss already knows.
  • Evidence dumps make phone review impractical.
  • Analogies dilute concrete findings.

Fix / Recommendation

  • Lead with one verdict.
  • Limit findings to four bullets.
  • Limit recommendations to four bullets.
  • Link evidence instead of pasting it.
Decision: approve only when the whole chart fits on one screen.
