Content Use Avoid

Option tradeoffs Side-by-side columns Unranked brainstorm

Quality checks

Required

  • One fixed Perch stylesheet
  • No inline style attributes or style blocks
  • No horizontal overflow at nested widths
  • Readable phone layout
  • Concrete final action or open question

Useful restraint

  • Stay under roughly two screens
  • Use one diagram only when relationships need it
  • Keep status badges short
  • Do not expose tokens or private paths
  • Do not turn a simple answer into an artifact
