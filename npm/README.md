# Perch (`perchctl`)

Perch is a local-first mobile companion and durable orchestrator for Claude Code and Codex agents running on your Mac.

The `perchctl` package installs the `perch` command.

```sh
npm install --global perchctl
perch doctor
```

Perch requires macOS, Node.js 20 or newer, npm 10 or newer, and an authenticated Claude Code or Codex CLI.

For complete setup, usage, architecture, operations, security, and other documentation, visit the [Perch repository on GitHub](https://github.com/zoidz123/perch).
